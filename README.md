# 💌 Хайрын захидал — Болзох санал

Интерактив "болзох санал" сайт. Хэрэглэгч нэр, асуулт, захидал, зураг, дуугаа оруулаад
хувийн **хайрын захидал** үүсгэж, линк/QR-аар хайртдаа илгээдэг. Хайрт нь нээхэд зугтдаг
"Үгүй" товчтой санал, "Тийм" дармагц **кино шиг баяр ёслол** (зүрх, цэцгийн дэлбээ,
гар бичмэл захидал, хөгжим) эхэлнэ.

🌐 **Амьд сайт:** https://hangvv11.github.io/bolzoo/

---

## ✨ Боломжууд

- **Бэлдэх редактор** — баруунд бичихэд зүүнд **шууд урьдчилан харагдана** (Санал / Баяр ёслол таб)
- **Зугтдаг "Үгүй" товч** — дарах гэхэд зайлж, хэзээ ч баригдахгүй (дэлгэцнээс гарахгүй)
- **Кино шиг баяр ёслол** — цохилох зүрх, нэр script бичгээр, зураг polaroid болж унах,
  захидал typewriter-ээр бичигдэх
- **💳 Демо төлбөр** — QPay QR / Карт / Банкны шилжүүлэг (жишээ, бодит мөнгө биш)
- **🔗 Хуваалцах** — линк + QR; **зураг шахагдаж линкэнд багтана**
- **✏️ Засах** — `&edit=1` тугтай "засварлах линк"-ээр дахин засаж болно (хайртад нь нуугдана)
- **🎵 Хөгжим** — YouTube дуу холбож болно, эсвэл файлгүй **нийлэгжүүлсэн уянгын аялгуу**
  (реверб, дулаан pad, дуулах lead)

## 🛠️ Хөгжүүлэлт

Бүх юм нэг файлд: **`index.html`** (HTML + CSS + JS). Build, dependency, сервер шаардлагагүй.
Зүгээр browser-т нээгээд засна.

### Өөр компьютер дээр үргэлжлүүлэх

```bash
# 1. GitHub-д нэвтрэх (нэг удаа)
brew install gh                 # Mac. Windows: https://cli.github.com
gh auth login --git-protocol https --web

# 2. Repo-г татах
git clone https://github.com/HANGVV11/bolzoo.git
cd bolzoo
```

### Өдөр тутмын ажил

```bash
git pull                        # эхлэхдээ — сүүлийн өөрчлөлтийг авах
# ... index.html-ээ засах ...
git add -A
git commit -m "тайлбар"
git push                        # → GitHub Pages 1-2 мин дотор шинэчлэгдэнэ
```

> 💡 Хурдан засвар: github.com → repo → `index.html` → ✏️ дарж browser дотроос шууд засаж болно.

## ⚠️ Хязгаарлалт

- **Төлбөр демо** — бодит мөнгө шилжихгүй. Жинхэнэ болгох бол QPay/Stripe + backend хэрэгтэй.

  тайрч магадгүй. Цөөн зураг найдвартай.
- **YouTube дуу** — интернэт шаардлагатай; зарим видео embed/бүс хязгаартай байж болзошгүй.
